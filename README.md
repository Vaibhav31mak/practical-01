# 📌 Calculator (.NET 10 Console App)

## 🚀 Overview
A simple console-based calculator that performs basic arithmetic operations on two integers.

## 🎯 Objective
- Practice input/output handling in .NET
- Understand basic arithmetic operations
- Build console interaction flow

## 🛠️ Tech Stack
- Language: C#
- Framework: .NET 10
- Type: Console Application

## 🧠 Concepts Covered
- Console input/output
- Data types (int)
- Arithmetic operations
- Basic validation

## ⚙️ How It Works
1. User enters two integers
2. Program performs:
   - Addition
   - Subtraction
   - Multiplication
   - Division
3. Results are displayed with clear messages

## ▶️ How to Run
1. Clone the repository
2. Navigate to project folder
3. Run:
   ```bash
   dotnet run
